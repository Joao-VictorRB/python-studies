# 🐍 Python Studies

Repositório dedicado aos meus estudos em **Python**, reunindo exercícios, desafios e pequenas aplicações desenvolvidas ao longo do **Curso em Vídeo de Python**, ministrado pelo professor **Gustavo Guanabara**, além de exercícios da disciplina de **Estrutura de Dados** da graduação.

O objetivo deste repositório é consolidar os fundamentos da linguagem, documentar minha evolução e servir como base para projetos mais avançados nas áreas de **Engenharia de Dados**, automação e desenvolvimento de software.

---

## 📚 Conteúdo

### 🌎 Mundo 1 — Fundamentos

Nesta etapa são abordados os conceitos essenciais da linguagem Python.

- Configuração do ambiente (Python e PyCharm)
- Entrada e saída de dados (`input` e `print`)
- Operadores aritméticos
- Ordem de precedência
- Módulos e bibliotecas (`math`, `random`, etc.)
- Manipulação de strings
- Estruturas condicionais (`if` e `else`)

---

### 🌎 Mundo 2 — Estruturas de Controle

Nesta etapa são exploradas as principais estruturas de decisão e repetição.

- Condições aninhadas (`if`, `elif`, `else`)
- Estruturas de repetição com `for`
- Estruturas de repetição com `while`
- Interrupção de laços com `break`

---

### 🌎 Mundo 3 — Estruturas Compostas e Módulos

Conteúdos voltados à organização de código e estruturas de dados nativas do Python.

- Tuplas
- Listas
- Listas compostas
- Dicionários
- Funções
- Modularização
- Pacotes
- Tratamento de exceções (`try`, `except` e `finally`)

---

### 🧩 Data Structures

Pasta destinada aos exercícios e implementações desenvolvidos durante a disciplina de **Estrutura de Dados** da graduação.

Os conteúdos incluem:

- Listas
- Pilhas
- Filas

- Árvores
- Algoritmos de Busca
- Algoritmos de Ordenação

> **Observação:** Esta seção é atualizada continuamente conforme novos conteúdos são estudados durante a graduação.

## ⚙️ Pré-requisitos

Antes de executar os exercícios, é necessário possuir:

- Python 3.x
- Editor de código (VS Code, PyCharm ou outro de sua preferência)

Para verificar se o Python está instalado:

```bash
python --version
```

ou

```bash
python3 --version
```

---

## ▶️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/Joao-VictorRB/python-studies.git
```

Entre na pasta:

```bash
cd python-studies
```

Execute qualquer exercício:

```bash
python caminho/do/arquivo.py
```

Exemplo:

```bash
python World1/ex001.py
```
